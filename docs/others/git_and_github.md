# Git/Githubについて

## ブランチの命名
```
[カテゴリ]/#[issue番号]-[ブランチ名]
```
### カテゴリの種類
| ブランチの種類 | 説明                           |
| ------------- | ------------------------------ |
| feature       | 新しい機能の開発                |
| bugfix        | バグや不具合の修正              |
| docs          | ドキュメントの追加や修正        |

**例1**：機能追加でメモ作成のAPIを作成する場合
```
feature/#40-create-memos-api
```

**例2**：開発フローのドキュメントを作成する場合
```
docs/#46-dev-flow-docs
```
